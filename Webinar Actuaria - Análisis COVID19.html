Contagios Acumulados

Esto es un poco mÃ¡s avanzado, pero se puede obtener lo mismo de arriba de manera mÃ¡s rÃ¡pida y para todos los paÃ­ses.

base_glob<-coronavirus %>% 
  filter(date == (date)) %>%
  select(country = country, type, cases,date) %>%
  group_by(country, type,date) %>%
  summarise(total_cases = sum(cases)) %>%
  pivot_wider(names_from = type,values_from = total_cases) %>%
  arrange(country,date) %>%
  mutate(date = as.Date(date), confirmados_acum = cumsum(confirmed), 
         recuperados_acum= cumsum(recovered), 
         muertos_acum= cumsum(death))
